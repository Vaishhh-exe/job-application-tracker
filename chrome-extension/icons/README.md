# Chrome Extension Icons

The extension requires icons in the following sizes:
- icon16.png (16x16)
- icon32.png (32x32)  
- icon48.png (48x48)
- icon128.png (128x128)

## Creating Icons

You can create these icons using any image editing software or online tools:

### Option 1: Use an Online Icon Generator
- Visit https://favicon.io/favicon-generator/ or similar
- Create a simple icon with "M" or "📝" emoji
- Download and resize to required dimensions

### Option 2: Simple Text Icons
Create simple colored squares with the letter "M" for Meridian:
- Background: #667eea (purple-blue gradient)
- Text: White "M" 
- Font: Bold, centered

### Option 3: Use Existing Favicons
If you have favicons for your Meridian app, convert them to the required sizes.

### Temporary Solution
The extension will work without icons, but Chrome will show a default puzzle piece icon instead.
